## Mimiflow - Japannese Learning Audio Repeater

A modern, web-based audio repeater application built with Next.js and React. Designed specifically for Japannese learners, this app provides an interactive listening experience with precise text-to-audio synchronization, sentence looping, and a focus-enhancing blind listening mode.

### ✨ Key Features

Interactive Audio-Text Sync: The text automatically highlights in real-time as the audio plays, keeping your eyes perfectly synced with your ears.

Single-Sentence Looping: Click the loop icon on any sentence to repeat it endlessly until you master the pronunciation and rhythm.

Blind Listening Mode: Blurs the transcript to test your pure listening comprehension. The text only reveals itself when it's actively playing or when you hover over it.
